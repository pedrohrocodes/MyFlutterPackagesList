## Flutter Social Signin Buttons

**Link:**
[flutter_signin_button](https://pub.dev/packages/flutter_signin_button)

**Examples:**
```
SignInButton(
  Buttons.Google,
  onPressed: () {},
)  
```

```
// with custom text  
SignInButton(
  Buttons.Google,
  text: "Sign up with Google",
  onPressed: () {},
)
```

```
SignInButton(
  Buttons.Facebook,
  mini: true,
  onPressed: () {},
)
```
